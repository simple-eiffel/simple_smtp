## Protocol Flow

```
Client                              Server
   |                                   |
   |-------- Connect (TCP) ---------->|
   |<------- 220 Ready ---------------|
   |-------- EHLO hostname ---------->|
   |<------- 250 OK ------------------|
   |-------- AUTH LOGIN ------------->|
   |<------- 334 Username ------------|
   |-------- base64(user) ----------->|
   |<------- 334 Password ------------|
   |-------- base64(pass) ----------->|
   |<------- 235 Auth OK -------------|
   |-------- MAIL FROM:<...> -------->|
   |<------- 250 OK ------------------|
   |-------- RCPT TO:<...> ---------->|
   |<------- 250 OK ------------------|
   |-------- DATA ------------------->|
   |<------- 354 Start -------------- |
   |-------- headers + body + . ----->|
   |<------- 250 OK ------------------|
   |-------- QUIT ------------------->|
   |<------- 221 Bye -----------------|
```
